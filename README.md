# Telegram-бот для инвентаризации инструмента

Бот хранит данные в SQLite и показывает меню из 4 пунктов:

1. **Список всех инструментов** (для всех)
2. **Инструменты + порядковый номер** (для всех)
3. **Объекты → инструменты на объекте** (для всех)
4. **Перемещение инструмента между объектами** (для сотрудников, с журналом)

Админка доступна администратору по команде **`/admin`** (редактирование справочников, списание/возврат, импорт/экспорт).

## Быстрый старт (Windows / PowerShell)

1) Создайте виртуальное окружение и поставьте зависимости:

```bash
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

2) Создайте файл `.env` в корне проекта:

```env
BOT_TOKEN=123456:ABCDEF...
ADMIN_IDS=11111111,22222222
DB_PATH=D:\inventar\inventory.sqlite3
```

3) Запуск:

```bash
python -m app
```

## Данные

- Инструмент имеет: **наименование**, **порядковый номер** (например `0001`), **статус** (в работе/списан), **текущий объект**.
- Перемещения пишутся в журнал.
